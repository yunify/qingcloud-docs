QingCloud API 网关根据 QingCloud AppCenter 功能特点，提供`基础版`、`专业版`、`企业版`、`企业高级版`四种专享实例规格，实例规格详情如下表。

> **说明**
>
> 专享实例的所有云资源均为单用户独享，较适合大中型企业用户的正式生产环境。
