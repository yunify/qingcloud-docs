本小节为您介绍如何升级高防 IP 规格，帮助您在线修改高防 IP 实例配置。

## 操作步骤

1. 登录管理控制台。

2. 在控制台导航栏中，选择**产品与服务** > **安全服务** > **高防IP**，进入**高防 IP-实例列表**页面。

3. 进入**实例列表**页面，在末尾**操作**栏选择**升级**。

   ![](../../_images/ddos_upgrade.png)

4. 选择需要升级的实例规格。

   > 仅支持修改基础防护带宽、最大防护带宽和回源带宽。

   ![](../../_images/ddos_upgrade_01.png)

5. 点击**提交**。
