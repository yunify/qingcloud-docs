##  云服务器挂载硬盘类型

| 云服务器类型                | 挂载硬盘类型         |
| ------------------- | -------------- |
| 基础型             | 基础型本地盘、容量型云盘、通用型 SSD 云盘和增强型 SSD 云盘。 |
| 企业型 | SD本地盘、容量型云盘、通用型SSD云盘和增强型SSD云盘。 |
| 专业型（GPU 云服务器） | 企业级SSD硬盘，数据盘支持企业级SSD硬盘、容量型和企业级分布式 SAN |

##  云服务器和硬盘区域（zone）

云服务器需要与硬盘所属同个可用区，如 云服务器（北京3B） 硬盘也必须为北京3B ；

## 副本策略

云服务器副本策略是否与硬盘副本策略一致；   

例：云服务器副本策略为同数据中心多副本 — 硬盘副本策略也需一致。
