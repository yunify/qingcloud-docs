请注意，只有当主机位于自管私有网络之中时，您才需要自行配置 IP 地址。

1. **开始** > **控制面板** > **网络**。
2. 右键点击 **本地连接**，选择 **属性**。
3. 在弹出的对话框中，选中 **Internet 协议版本4**。
4. 点击 **属性** 按钮， 在弹出的对话框中即可设置静态 IP 地址。
