本文主要介绍用户在共享集群模式和专属集群模式下，涉及的计费资源及对应的计费方式。

## 计费模式

弹性高性能计算 EHPC 支持**包年/包月**和**按需计费**两种计费模式。

| <span style="display:inline-block;width:100px">计费模式</span> | 说明                                                         | 适用场景                                                     |
| :----------------------------------------------------------- | :----------------------------------------------------------- | :----------------------------------------------------------- |
| 包年/包月                                                    | **计费方式**为`月`或`年`。您需要按照购买时长（月或年）一次性支付所选时长的费用。在购买时长期间内，您可以一直使用该资源。 | 适用于长期稳定需求，帮助您更大程度的节省支出。               |
| 按需计费                                                     | **计费方式**为`小时`。按秒计费，按小时扣费，且不设最低消费指标，您可以随时启用和删除资源。<br />若集群实际使用时长不足 1 小时，将按实际时长进行扣费。<ul><li>集群开启时，收取节点主机实例和节点存储空间费用。</li><li>集群关闭时，仅收取节点存储空间费用。</li></ul> | 适用于有较大波动且无法准确预测资源需求量的业务场景，或临时性和突发性的资源需求场景。如果是短期测试使用，推荐使用按需计费模式。 |

## 计费项

弹性高性能计算 EHPC 仅对集群所占用的资源进行收费。

- 共享集群只需要一个登录节点，连接到共享计算队列即可。

- 专属集群需要全部节点。

| 计费项          | 计费说明                                                     |
| --------------- | ------------------------------------------------------------ |
| 集群节点主机    | 根据您选择的节点主机规格和数量进行收费。                     |
| CPU 核心数      | **共享集群**，提交作业时作业需要用到的核心数，系统将根据核心数和时长进行计费。 |
| 文件存储 EPFS   | 按申请容量进行收费。                                         |
| VPC 资源        | 您可以使用免费的 VPC 资源。若您使用收费的 VPC 资源，价格详情请参见 [VPC 网络计费](/network/vpc/billing/price/)。 |
| 公网 IP（可选） | 若您绑定了公网 IP，公网 IP 产生的费用将会另行计费，价格详情请参见[公网 IP 计费说明](/network/eip/billing/price/)。 |

## 共享集群-收费示例

用户在共享集群模式下，产生的费用包括：

- 集群资源费用：登录节点费、作业核时费。
- 文件存储费用：按资源容量和时长进行收费。
- 网络资源费用：VPC网络，VPC 按型号收费。公网IP，按带宽或流量收费。

例如：用户需要应用共享集群进行作业，预计需要 10000 核时，登录节点1核2G，存储2T。公网 IP 按 5M 带宽计费。

## 专属集群-收费示例

用户在专属集群模式下，产生的费用包括：

- 集群资源费用：登录节点、管控节点、计算节点。按各个节点的配置类型和使用时长进行计费。

- 共享存储费用：按资源容量和时长进行收费。

- 网络资源费用：VPC网络，VPC 按型号收费。公网IP，按带宽或流量收费。

  > **说明：**
  >
  > 功能测试用小集群即可，性能测试和大作业用超级计算节点。

例如：用户需要创建1个集群、4个计算节点、配置是32核 240G、登录节点1核2G、管控节点2核4G、存储2T。公网 IP 按 5M 带宽计费。
