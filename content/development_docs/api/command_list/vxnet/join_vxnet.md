将主机加入到私有网络。当需要主机加入私有网络时指定 IP 地址，只支持单台主机加入。

警告

一台主机最多只能加入一个受管网络 ( 包括基础网络vxnet-0 )

**Request Parameters**

| Parameter name | Type | Description | Required |
| --- | --- | --- | --- |
| vxnet | String | 私有网络 ID (如果要指定 ip 地址，格式为'私有网络 ID\|IP 地址') | Yes |
| instances.n | String | 加入到网络的主机 ID | Yes |
| zone | String | 区域 ID，注意要小写 | Yes |

[_公共参数_](../../../parameters/)

**Response Elements**

| Name | Type | Description |
| --- | --- | --- |
| action | String | 响应动作 |
| job_id | String | 执行任务的 Job ID |
| ret_code | Integer | 执行成功与否，0 表示成功，其他值则为错误代码 |

**Example**

_Example Request_:

```
https://api.qingcloud.com/iaas/?action=JoinVxnet
&vxnet=vxnet-q8f2bu7
&instances.1=i-opv1n2brh
&instances.2=i-m163jcqw1
&zone=pek3a
&COMMON_PARAMS
```

_Example Response_:

```
{
  "action":"JoinVxnetResponse",
  "job_id":"j-pp51vko0",
  "ret_code":0
}
```
