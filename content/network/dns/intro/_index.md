## DNS服务

域名系统（Domain Name System，DNS）是互联网的一项基础网络服务，将域名和 IP 地址相互解析映射，以助用户便捷地访问互联网。

QingCloud DNS 服务是一款权威域名解析服务，提供可视化管理域名智能解析，并跟踪域名解析情况。

> DNS 服务正处于免费公测期间，如遇到任何问题可通过[提交工单](https://console.qingcloud.com/tickets/#)与我们联系。
