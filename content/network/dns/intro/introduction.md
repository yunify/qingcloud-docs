## DNS服务

域名系统（Domain Name System，DNS）是互联网的一项基础网络服务，将域名和 IP 地址相互解析映射，以助用户便捷地访问互联网。

QingCloud DNS 是一款权威域名解析服务，提供可视化智能域名解析管理，并跟踪域名解析情况。
