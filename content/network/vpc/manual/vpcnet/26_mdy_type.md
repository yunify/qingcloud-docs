## 操作场景

当已创建 VPC 网络已不适用您当前的网络规模，您可以对 VPC 网络进行规格升级或降级。

> **说明**：
>
> 免费 VPC 不支持升级为其他类型。其他类型的 VPC 若更改为免费 VPC，将不能绑定公网 IP 和转发公网流量。若您需要通过 VPC 的公网 IP 从公网访问端口转发、隧道服务（GRE 隧道、IPSec 隧道）、VPN 服务等管理服务，则不建议您更改为免费类型。



## 操作步骤

1. 登录[管理控制台](https://console.qingcloud.com/pek3)。

2. 在控制台导航栏中，选择**产品与服务** > **网络服务** > **VPC网络**，进入**VPC网络**页面。

3. 找到需要更改的 VPC 网络，点击**更多操作** > **扩容**。

   <img src="/network/vpc/_images/501026_mdy_type.png" alt="501026_mdy_type" style="zoom:50%;" />

4. 在弹出的页面中，修改类型，点击**提交**。
