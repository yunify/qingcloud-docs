本文介绍 API 请求的组成及示例说明。

## 服务器地址

青云云点播 API 的接入域名为： `api.vod.frontwize.com`。

## 通信协议

青云云点播 API 的所有接口均通过 **HTTPS** 进行通信，提供高安全性的通信通道。

## 请求方法

HTTP请求方法告诉服务你正在请求什么类型的操作。

青云云点播 API 支持的 HTTP 请求方法如下：

| 方法   | 说明                               |
| ------ | ---------------------------------- |
| POST   | 请求服务器新增资源或执行特殊操作。 |
| GET    | 请求服务器返回指定资源。           |
| PUT    | 请求服务器更新指定资源。           |
| DELETE | 请求服务器删除指定资源。           |

## 字符编码

请求及返回结果都使用`UTF-8`字符集进行编码。
