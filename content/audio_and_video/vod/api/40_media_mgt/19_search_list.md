调用 **SearchMediaList** 查询媒资详情，如媒资基本信息、转码信息、审核结果信息等。

## 请求路径

/v1/media/search

## 请求方法

GET

## 请求参数



## 返回参数

**状态码： 200**

| 参数      | 参数类型                   | 描述                                                         |
| :-------- | :------------------------- | :----------------------------------------------------------- |
| Total     | Integer                    | 媒资总数说明：暂只能统计xxx个媒资，若您需要查询具体的媒资总数，请提交工单申请。  <!--待确认--> |
| MediaInfo | Array of mediainfo objects | 媒资详情列表。                                               |
