调用 **ReleaseMedia** 将禁播媒资解禁。

## 请求路径

/v1/media/\<MediaId\>/release

## 请求方法

POST

## 请求参数

| 参数  | 是否必选 | 参数类型 | 描述   |
| ----- | -------- | -------- | ------ |
| AppId | 否       | Integer  | 预留。 |



## 返回参数

**状态码： 200**
