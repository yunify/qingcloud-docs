视频直播提供音视频采集、直播转码、多边缘节点加速等。

| 功能          | 功能说明                                                     |
| ------------- | ------------------------------------------------------------ |
| 推流协议      | RTMP                                                         |
| 播放协议/格式 | RTMP、HTTP flv、HTTP m3u8                                    |
| 域名管理      | 支持编辑域名，您可以添加推流域名和播放域名。添加成功后，系统自动生成 CNAME 记录。 |
| 直播监控      | 用户可以通过直播监控功能，查看对应域名的流量与带宽使用情况。 |
