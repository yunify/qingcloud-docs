数据源定义关系型数据库、NoSQL 数据库、半结构化存储、消息队列等多样性数据连接的信息，用于数据同步、云上加工等开发步骤。您可以在数据源管理页面查看、新增及修改数据源。

数据源需要在连通网络的前提下进行数据同步，详情请参见[网络连通方案](../../connect/)。

支持的数据源如下：

- MySQL
- PostgreSQL
- ClickHouse
- HBase
- Kafka
- Ftp
- HDFS
