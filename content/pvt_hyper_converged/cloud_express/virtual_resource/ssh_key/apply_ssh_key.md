本小节主要介绍如何应用 SSH 密钥到主机。

## 操作步骤

1. 登录 青立方® 超融合易捷版。
2. 选择 **虚拟资源** > **SSH 密钥**，进入密钥管理列表。
3. 勾选目标密钥，点击**加载到主机**，弹出选择主机窗口。
4. 选择一台主机。
   
   ![选择主机](../../../_images/ssh_host.png)

5. 确认配置信息无误后，点击 **确定**，返回密钥列表页面

   ![选择主机](../../../_images/ssh_host2.png)
