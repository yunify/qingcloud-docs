## 默认用户

创建 TiDB 实例时，将同步创建一个默认用户：**root**，默认无密码，可在实例创建后设置及修改 root 用户密码。

> **说明**
>
> - 为了保障信息安全，建议您创建后立即设置密码。具体操作方法请参见[修改用户密码](../mdypasswd/)。
> - root 用户不支持修改用户及权限。
