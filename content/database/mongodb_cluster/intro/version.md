MongoDB Cluster 基于 AppCenter 功能特点，提供更稳定更高性能 MongoDB 数据库服务。

## 版本历程

### 4.0.3 - v1.0.0

MongoDB Cluster 4.0.3 - v1.0.0 基于 MongoDB 4.0.3 内核构建，集成 Zabbix Agent2（5.4）。

- 具有高性能、高可用、可扩展等优点
- 支持 Ranged 和 Hashed 两种分片策略
- 支持全量物理备份
- 支持海量数据的横向扩展
- 支持多可用区部署，提供全量物理备份功能
