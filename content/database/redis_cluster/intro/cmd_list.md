不同版本的 Redis Cluster 实例对 Redis 命令的支持度有所不同。

## 注释与说明

为便于阅读和内容表达，本文的表格约定使用下述注释：

- **✓** 表示支持该命令。
- **✕** 表示在不支持该命令。
- **⎻** 表示在原生Redis的该版本下，该命令尚未开始支持。
- **config** 表示支持该命令，但需要在控制台修改控制参数支持。
